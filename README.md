# Daily Motivation WhatsApp Agent (n8n)

An AI-powered automation workflow built using n8n, Groq LLM, and Twilio WhatsApp API.

## Features

- Sends a unique motivational quote daily
- Runs automatically on schedule
- Uses AI to generate fresh quotes every day
- Delivers to WhatsApp instantly

## Stack

- n8n
- Groq API
- Twilio WhatsApp API

## Workflow Logic

Schedule Trigger → AI Quote Generator → WhatsApp Sender

## Use Cases

- Personal productivity
- Daily reminders
- Motivation bots
- AI automation portfolio project
